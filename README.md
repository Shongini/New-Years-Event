# Sylwester Event Manager

A simple desktop application for managing New Year's Eve event guests and information, built with Python and [Flet](https://flet.dev/).

## Features
- **Guest Management:** Add, delete, and mark guests as "Paid" or "Unpaid".
- **Event Information:** View event location, price per person, and list of organizers.

## Prerequisites
- Python 3.8+
- [Flet](https://flet.dev/) library

## Installation
1. Clone this repository:
   ```bash
   git clone https://github.com/Shongini/New-Years-Event
   cd New-Years-Event
   ```
2. Install the required dependencies:
   ```bash
   pip install flet
   ```

## How to Run
Run the application using the following command:
```bash
python main_app.py
```

---
*Note: This project uses a `data.json` file for local storage of guest information.*
